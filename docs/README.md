## 项目概述

OGScope 是一个基于 Orange Pi Zero 2W 的电子极轴镜系统，专为天文摄影爱好者设计。

### 主要特性

- 🔭 **精确校准**: 高精度极轴校准算法
- 📱 **远程控制**: Web 界面和移动 App
- 🖥️ **本地显示**: 2.4寸 SPI LCD 实时显示
- 🌐 **生态集成**: 支持 INDI 协议

### 技术规格

- **处理器**: Orange Pi Zero 2W (Allwinner H618)
- **相机**: IMX327 传感器 (1920x1080)
- **显示**: 2.4寸 SPI LCD (240x320)
- **软件**: Python 3.9 + FastAPI
