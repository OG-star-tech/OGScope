WiFi 引导

  1. 手机/电脑连接热点 OGScope_xxxx，密码 ogscopeadmin
  2. 浏览器打开本页（AP 下地址一般为 http://192.168.4.1:{{ http_port }}）
  3. 点击下方「扫描 WiFi」（由设备执行 nmcli，浏览器无法直接扫 WiFi）；热点(AP)模式下部分网卡列不出周边网络，请用手动输入 SSID
  4. 连接成功后设备会加入局域网；可用下方「mDNS」或「扫描局域网」找回新地址（需同网段 HTTP，HTTPS 页面无法扫 HTTP）
